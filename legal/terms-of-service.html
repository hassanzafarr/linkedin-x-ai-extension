2. Description of Service

EngageFlow AI is a Chrome browser extension that:

  • Reads visible content from LinkedIn and X (Twitter) pages you browse
  • Sends that content to the Anthropic Claude API using your API key to generate AI-powered reply suggestions and post drafts
  • Optionally reads your own LinkedIn profile to build a writing voice profile
  • Stores drafts and settings locally in your browser

The extension is a standalone client-side tool. It has no EngageFlow AI backend, account system, or subscription service.
